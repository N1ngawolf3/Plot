1. Скопировать папку себе *Один раз в начале, больше не надо
git clone https://github.com/N1ngawolf3/Plot.git
*************Процесс добавления изменений***************************
1. Добавить измененные файлы
git add *
КОПИРОВАТЬ СО ЗВЁЗДОЧКОЙ
2. Подтвердить изменения
git commit -m 'Change'
Вместо change на английском написать что изменили, кратко, вместо с кавычками
3. Отправить изменения
git push
Если ошибок нет, поздравляю, если есть - ебитесь
